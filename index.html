Welcome to the ARAD Skill Planner CSS Page

You can download a copy of the CSS files here to make a theme for the planner, which you can then submit to me or Aquarium.

Why?

Because I feel like it would be nice to potentially have user contributed themes to the site.

How?

I suggest you know at least a little CSS before trying to create a theme! I have provided a list of CSS IDs for your use, however. Check the support section (located below) if you need help. https://github.com/DarkstarIV/DFO-Planner-CSS/wiki/CSS-IDs

Authors and Contributors

Aquarium for the planner itself, and the base style css. Darkstar (me) created the dark style.

Support or Contact

Having trouble styling it? Send me a PM on any of the following sites for help.

-DFO Source (DarkstarIV) -Super Sword Action (Darkstar) -Github (DarkstarIV)

https://github.com/DarkstarIV/DFO-Planner-CSS/issues
